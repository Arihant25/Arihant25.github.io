About this project

Built for the Wikimedia Hackathon 2024, this dashboard helps Wikimedia community members monitor unusual website traffic and take action, such as organizing events or analyzing impact. I developed the website while my collaborator, Navya Sri Kalli, handled the Python data analysis.
