About this project

This was an assignment for the Design and Analysis of Software Systems Spring 2025 course at IIIT-H. We had to make a diet manager app with several features like calorie tracking, meal database management, daily food logs, etc. We also had to make documentation for it in terms of UML diagrams. This was made in collaboration with Mohit Kumar Singh.
